# Implementation Plan 9: QR Package

**Parent plan:** [ziron_tap_structure_plan_43dd61bf.plan.md](.cursor/plans/ziron_tap_structure_plan_43dd61bf.plan.md)

**Prerequisites:** Plan 7 (Media for logo overlay)

---

## Scope

- Multi-type QR: url, pdf, socials, file, contact, plain_text, wifi, map_location, nfc
- Encode payloads (vCard, WiFi string, geo URL)
- Render SVG/PNG with logo, colors, margin, error level
- Layout presets (square, rounded, with label)

---

## Key Files


| File                         | Purpose                             |
| ---------------------------- | ----------------------------------- |
| `packages/qr/src/encode.ts`  | Payload derivation per type         |
| `packages/qr/src/render.ts`  | SVG/PNG rendering with logo, colors |
| `packages/qr/src/types.ts`   | Type enum + Zod validation schemas  |
| `packages/qr/src/presets.ts` | Layout presets                      |


---

## Implementation Steps

1. **Types** — Define QR type enum, payload schemas, customization options
2. **Encode** — vCard for contact, WiFi string, geo URL for map_location, etc.
3. **Render** — Use qrcodegen or similar; support logo overlay from @ziron/media
4. **Presets** — Square, rounded, with label layouts

---

## Dependencies

- `qrcodegen` or similar
- `@ziron/media` for logo overlay
